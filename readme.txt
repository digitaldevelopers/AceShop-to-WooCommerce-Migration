This plugin is a hand of help for migration from AceShop to WooCommerce with no technical skills required.

== Description == 
Сart2Cart: AceShop to WooCommerce Migration Plugin gives an opportunity to get your stores connected and provide an effortless data move. Migration is automated and quick, offering an accurate and safe move of all the information (products, orders, customers, etc.). This plugin is costless, but still you will have to pay for Full migration. The price depends on an amount of selected transfer items and starts from  $49. 


= Features of automated migration =
- **Simple** - Migration Wizard will offer a couple of automated steps that are necessary to make. 
- **Fast** - you need only a few hours to complete the migration.
- **Free Demo** - check service functionality within 10-30 minutes.
- **Support** - et an answer for any question by phone, via chat or email.


= You can migrate the following data from AceShop to WooCommerce =
* Products, product images, product extra fields, product attributes;
* Categories, category images
* Customers, shipping address, billing address;
* Orders, order statuses.

*Supported AceShop versions:* 1.x, 2.x, 3.x
*Supported WooCommerce versions:* 1.4.x, 1.5.x, 1.6.x, 2.0.x (new software versions are constantly being added).

**Please note.** *This plugin provides gives a possibility to move data only, so the template is not going to be migrated.*

Cart2Cart: AceShop to WooCommerce Migration offers a Connection bridge installation for both web stores. So that the connection between them will be set and data interaction provided. After that you are going to be redirected to Cart2Cart’s official page and will get a chance to continue your migration.

= Steps to take before migration =
* Install WooCommerce and check online availability of your AceShop and WooCommerce stores.
* Prepare AceShop FTP access information as you will need them for Connection bridge installation. 

==Installation==
1. Download the plugin zip file.
1. Extract plugin archived file to your computer.
1. Upload unzipped file to wp-content/plugin directory.
1. Go to Admin -> Plugins, proceed to “Cart2Cart Magento to WooCommerce Migration” and press Activate.
1. Paste AceShop FTP details and download Connection bridge files. 
1. An upload of your WooCommerce connection bridge will be performed automatically.
1. Click Start Migration and wait for a redirection to Cart2Cart site to finish migration procedure.

 
== Frequently Asked Questions ==
= What is the estimation of the migration time? = 
Migration Estimator allows to calculate not only the cost, but also the approximate time of migration performance.
We can not provide you the exact estimates of migration time as it depends on each particular migration case. The time of your data migration will depend on: your server peculiarities, type of shopping carts you migrate from and to, the quantity of items for migration, type of store data.
Migration estimator shows the average time necessary for the quantity of entities you specify, not considering the above mentioned details. So don't expect the migration to complete exactly in the time, previously estimated
Please, be sure that we do everything that depends on us to provide you with the highest speed of migration performance.

= Can I be sure of data security Cart2Cart service? = 
Cart2Cart performs migration using a separate Amazon EC2 server and all data is deleted after the process has been finished.
Cart2Cart does not migrate client’s credit card information and passwords.
We use secure HTTPS protocol for all connections and only authorized staff has access to the migration process.

= What does the option Clear current data on Target store before migration mean? =
Additional option “Clear current data on a Target store before Migration” allows to delete data from your target store based on selected entities. It means if you’ve selected products and taxes migration on a step Entities Selection of Migration Wizard, with this option on a target store will be deleted only products and taxes.
In a case you’ve decide to perform multi-store migration, with this option will be deleted data from a store you are moving to.
Note! This option is irreversible, so it is highly recommended to make a backup before migration.

= Can I migrate payment methods and invoices? =
Unfortunately, Cart2Cart does not support migration of payment methods and invoices.

= Default language configured incorrectly =
Make sure your ISO code is specified properly.
You may also install the language you need and set it as the default language of the store.
This procedure could be performed via the admin panel of your shopping cart.
